## 常用的排序算法学习


### 冒泡排序

#### 介绍
数组中相邻的元素进行对比，把符合规则的元素往后移动，会把符合规则的元素移到最后一位
重复上述动作直至排序完成


#### 示例过程

做一个升序排列
原始值：[3,2,1]
[2,1,3]
[1,2,3]

最终的结论就是： 外层循环是 length-1  内层循环是 length - 1 - i
